## Remarks
Reserved words are keywords used by blast to identify built in (flow)control operations: 
```
Flow control:
    IF THEN ELSE
    SWITCH CASE DEFAULT    
    WHILE | FOR

Compiler control:
    DEFINE 
    INPUT
    OUTPUT
    VALIDATE

And any defined function name.
```
